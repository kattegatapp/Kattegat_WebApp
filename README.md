# Kattegat WebApp

Next.js App Router web app for the Kattegat launch waitlist and admin entry point.

## Stack

- Next.js `16.2.10` App Router
- React `19`
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- TanStack Query
- TanStack Table
- React Hook Form
- Zod
- Lucide React
- Recharts

## Design System Rule

**shadcn/ui is the primary design library for this web app.**

Use shadcn components from `src/components/ui` and install missing primitives with:

```bash
npx shadcn@latest add <component>
```

Do not hand-build custom UI primitives when shadcn has the component. Product-specific
components should compose shadcn primitives internally, then apply Kattegat's theme,
glassmorphism, spacing, and brand assets on top.

The shadcn setup lives in `components.json`. Current installed primitives include:

- `button`
- `badge`
- `card`
- `input`
- `label`

## Kattegat Theme

The web app should match the mobile app's visual system:

- Forest `#003912`
- Mantis `#6FDB42`
- Emerald `#48DC81`
- Astronaut Blue `#1C4759`
- Plus Jakarta Sans
- Glassmorphic elevated surfaces

Mobile design source of truth:

```txt
../Kattegat_Mobile/src/theme
../Kattegat_Mobile/assets/images
```

Web brand assets live in:

```txt
public/brand
```

## Backend

The backend source of truth is:

```txt
../Kattegat_Backend
```

Use backend contracts for real auth, admin tooling, billing, waitlist persistence, API
response envelopes, roles, permissions, and domain rules.

## Routes

- `/` - public launch waitlist
- `/kattegat-admin/login` - specialized admin login URL

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Commands

```bash
npm run dev
npm run lint
npm run build
npx shadcn@latest add <component>
```

## Folders

- `src/app` - App Router routes
- `src/components/ui` - shadcn/ui primitives generated by the CLI
- `src/components/waitlist` - public waitlist modules
- `src/components/admin` - admin login/dashboard foundation
- `src/lib` - providers, utilities, validation schemas
- `public/brand` - Kattegat brand assets shared from mobile
